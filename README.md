# Présentation

Voici TASS (Totally Accurate Battle Simulator) fortement inspiré de TABS mais version espace !
Dans ce jeu vous pourrez faire combattre des astres entre eux tous plus forts les uns que les autres entre astéroïdes, planètes, étoiles, trous noirs et autres :D

## Fun facts
- Ce jeu a été crée dans le cadre de la Aywen Game Jam du 4 avril au 4 mai 2026
- Ce jeu n'a aucun code écrit par l'IA, l'intelligence artificielle a seulement été utilisée pour des conseils, de la documentation ou des mathématiques
- Il n'est absolument pas terminé, la game jam étant toujours a ses débuts
